0. Введение

  • Наследование на уровне объектов в JavaScript реализуется через ссылку __proto__
  • Атрибуты классов хранятся в *.prototype.
  • В prototype по умолчанию всегда находится свойство constructor, указывающее на функцию-конструктор

1. Способы наследования

  • Rabbit.prototype.__proto__ = Animal.prototype;
  • Rabbit.prototype = new Animal();
  • Rabbit.prototype = Object.create(Animal.prototype);
Rabbit.prototype.constructor = Rabbit;

2. Вызов конструктора родителя

  Animal.apply(this, arguments);
}
